# Tasko - Task and Attendance Management System

A comprehensive enterprise management system built with Spring Boot.

## Features

- User Management
- Task Management
- Attendance Tracking
- Enterprise Management
- Payroll Management
- Analytics & Reporting

## Technology Stack

- Java 11
- Spring Boot 2.7.11
- Spring Security
- Spring Data JPA
- Thymeleaf
- H2 Database
- Maven

## Getting Started

1. Clone the repository
2. Build the project:
   ```bash
   mvn clean package
   ```
3. Run the application:
   ```bash
   java -jar target/tasko-0.0.1-SNAPSHOT.jar
   ```
4. Access the application at `http://localhost:8080`

## Available Features

### User Dashboard
- Task viewing and management
- Attendance logging (once per day)
- Profile management
- Navigation sidebar
- Real-time updates

### Admin Dashboard
- Enterprise management
- User management
- Payroll management
- Task oversight
- Analytics and reporting
- System monitoring
